# jBatch-study
jBatchのお勉強だよ。

## jBatchとは
[JSR 352](https://jcp.org/en/jsr/detail?id=352)で規定されています。

- Java EE 7で初めて導入されたバッチ処理の**標準仕様**(実装ではない)
- もともとSpringBatchが存在して、それを参考にして仕様化したのがjBatch

## jBatchの実装
- RI（GlassFish）
- JBeret（WildFly）
- SpringBatch
